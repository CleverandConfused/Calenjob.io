How they connect: Every page links to /about.html, /blog.html, and / from the navbar and footer. The blog list page links to each individual post. Add new blog posts as new flat files (e.g. 2026-mileage-deduction.html) and add a card to blog.html pointing at it.

Connection map

index.html (homepage)
  ├─ links to → about.html, blog.html
  └─ scroll to → #features, #pricing, #signup

about.html (the CalenJob story)
  └─ links to → index.html, blog.html

blog.html (post list)
  ├─ links to → index.html, about.html
  └─ links to each post → why-contractors-run-late.html

why-contractors-run-late.html (the post)
  └─ links back to → index.html, about.html, blog.html

How to add the next blog post

  1. Create a new file at the root, e.g. 2026-mileage-deduction.html
  2. Copy the structure of why-contractors-run-late.html as a template.
  3. Update the title, content, canonical URL, and OG tags to match the new post.
  4. Open blog.html, copy the existing post-card block, paste it above the existing one, and update href / title / excerpt / date.
  5. Move one of the "Coming Soon" cards to the published list when you publish each post.
